## 队列&单调队列

### 154. 滑动窗口

滑动窗口内O(1)求极值，则在滑动时维护单调队列（去除无用的数，发现具有单调性）

队列中存数组的**下标**：这样就可以在维护单调队列的同时，通过**下标差**记录和维护窗口大小k，，比较巧

```
 队头    队尾
[i-k+1,   i]  k大小窗口
```

对于输入样例，以求滑动窗口最小值为例，模拟一下单调队列变化过程

```
a[]: 1 3 -1 -3 5 3 6 7
q[]的随 i = 0 -> n-1 的变化（内容显示为存储的idx对应的a[idx]值）
           [head> ... tail>]
    i = 0:  1
    i = 1:  1 3
    i = 2: -1       输出 -1
    i = 3: -3       输出 -3
    i = 4: -3 5     输出 -3
    i = 5: -3 3     输出 -3
    i = 6:  3 6     输出  3
    i = 7:  3 6 7   输出  3
```
