T898 数字三角形
- 分析
  - 状态表示:【集合】从起点走到(i,j)的路径；【属性】max值
  - 状态计算：`dp[i][j] = max(dp[i-1][j-1] + a[i][j], dp[i-1][j] + a[i][j])`
- 主要就在给三角形标号
  ```
          (1,1)
      (2,1)  (2,2)
    (3,1) (3,2) (3,3)
          ....
  ```
实现：在dp数组初始化时出了好多nt的bug，比如：n读入前就使用n，`[i][i]`写成`[i][n]`，然而其实应该是`[i][i+1]`
